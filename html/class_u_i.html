string UI::point12 ( FileReader  x)
static

Calcula e retorna informações sobre o comprimento máximo da viagem.

Esta função obtém informações sobre o comprimento máximo da viagem a partir do objeto FileReader fornecido. Ela invoca a função getMaxStops da classe Statistics para encontrar o comprimento máximo da viagem, o número máximo de paragens e o número de aeroportos que compõem os caminhos de comprimento máximo. Em seguida, gera uma representação em string dessas informações para apresentação na interface do usuário.

Parameters
xObjeto FileReader contendo as informações necessárias para calcular o comprimento máximo da viagem.
Returns
Uma string que representa informações sobre o comprimento máximo da viagem, o número de paragens e os aeroportos que compõem os caminhos de comprimento máximo.
